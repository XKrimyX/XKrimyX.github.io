MarkDown语法笔记

1.标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
=====与-----分别代表一级和二级标题

2.格式

换行

使用两个以上空格加上回车来实现换行
段落后面使用一个空行来表示重新开始一个段落

字体

*斜体*
_斜体_
**粗体**
__粗体__
***粗斜体***
___粗斜体___

分割线

三个以上的星号、减号、底线为一个分隔线，可以在星号或减号中间插入空格

删除线

在文字的两端加上两个波浪线 ~~ 即可添加删除线

下划线

下划线可以通过 HTML 的 <u> 标签来实现

脚注

[^要注明的文本]

3.列表

有序列表

数字+”.” 号+空格+内容

  1. 一
  2. 二

无序列表

星号(*)、加号(+)或减号(-)+空格+内容

  • 一
  • 二

4.区块

段落开头使用 > 符号+空格+内容

5.代码

代码片段用反引号（`）把它包起来
代码区块用 4 个空格或一个制表符
或是用 ``` 包裹一段代码，并指定一种语言（也可以不指定）

6.链接

[链接名称](链接地址)

7.图片

![alt 属性文本](图片地址 “可选标题”)

8.表格

| 表头 | 表头 |
| —- | —- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |
-: 设置内容和标题栏居右对齐。
:- 设置内容和标题栏居左对齐。
:-: 设置内容和标题栏居中对齐。
